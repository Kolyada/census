# USA census

- Search American ZipCodes on zip, city, county, city abbreviation (autocomplete)
- Create group from zip manually or random or with nearest
- Look up all group objects at Google Maps 
- Other available data in **/db/result.csv**
- Scripts for load database are **/app/helpers/application_helper.rb loadRawData()**
- Description kind of data in **/config/app.yml** (russian)
- All data from [geodataservice.net](http://geodataservice.net) 
- Their policy - [here](http://geodataservice.net/PrivacyPolicy.aspx)
